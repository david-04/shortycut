Browser integration

This page contains links and tools for integrating ShortyCut with your browser. See the manual for a description of all methods.

Homepage

ShortyCut has settings to tweak the behavior of its homepage when used as the browser's start page or as a pinned tab. Activate them as required by ticking the checkboxes below:

  Open all links in new tabs
Lock ShortyCut's homepage to the browser tab and never navigate away. As the tab sticks around, it can be reused for entering more keywords later.
 
  Don't focus on the input field
Tick this checkbox when using an add-on like Vimium, which allows to operate the browser via keyboard shortcuts.

Search engine

To add ShortyCut as a search engine, it must be access through a web server. Follow the instructions in the manual to set it up. Once the web server is up and running, access ShortyCut through this link:

http://localhost:4171/index.html

For browsers which require an encrypted connection, use this link instead:

https://localhost:4172/index.html

This will produce a security warning. Confirm it to open ShortyCut.
ShortyCut should be discovered as a search engine automatically.
Please check your browser's th search box.

Pop-up blocker

ShortyCut allows multiple links / bookmarks to have the same keyword.
Depending on the configuration, it can open all of them in separate tabs.
To use this feature, you need to disable the browser's pop-up blocker.
Test if the blocker is currently active by clicking on this button:



If pop-ups are blocked, it's usually indicated in the browser's address bar.
You should be able to allow pop-ups for ShortyCut there as well.
